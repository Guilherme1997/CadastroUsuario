# FrontEnd

Os pacotes para a linguagem JavaScript deste projeto foram gerenciados com o npm.

Para obter o gerenciador de pacotes npm , faça o download do Node.js através do site:https://nodejs.org/en/download/

Este projeto foi gerado com o CLI angular.

Para obter a interface de linha de comando para o Angular, digitar o seguinte comando no prompt do
Windows: npm install -g @angular/cli.

# Servidor de Desenvolvimento 

Execute `npm i` na pasta EasyContact\FrontEnd para obter os pacotes necessários para este projeto.

Execute `ng serve` na pasta EasyContact\FrontEnd para um servidor dev. 

Navegue para `http://localhost:4200/`.

O aplicativo será recarregado automaticamente se você alterar qualquer um dos arquivos de origem.
